# API

```jsx
import { Badge } from 'kui-mobile';
```

## Badge

| 名称          | 类型             | 默认值    | 描述                                                     |
| ------------- | ---------------- | --------- | -------------------------------------------------------- |
| text          | number \| string | -         | 显示文本                                                 |
| overflowCount | number           | 99        | 最大显示数字                                             |
| dot           | boolean          | false     | 是否显示圆点                                             |
| color         | string           | 'primary' | 颜色，可选 'primary' 'info' 'success' 'warning' 'danger' |
