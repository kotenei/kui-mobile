# API

```jsx
import { LoadMore } from 'kui-mobile';
```

## LoadMore

| 名称    | 类型                      | 默认值 | 描述           |
| ------- | ------------------------- | ------ | -------------- |
| tip     | React.ReactNode \| string | -      | 提示           |
| loading | boolean                   | true   | 是否加载中     |
| full    | boolean                   | -      | 宽度是否百分百 |
