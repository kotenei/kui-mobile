# API

```jsx
import { Loading } from 'kui-mobile';
```

## Loading

| 名称     | 类型                      | 默认值 | 描述                                                           |
| -------- | ------------------------- | ------ | -------------------------------------------------------------- |
| color    | string                    | -      | 颜色风格，可选值 'primary' 'info' 'success' 'warning' 'danger' |
| size     | string                    | -      | 尺寸，可选 'sm' 'lg'                                           |
| tip      | React.ReactNode \| string | -      | 加载提示                                                       |
| vertical | boolean                   | false  | 是否垂直                                                       |
