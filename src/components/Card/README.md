# API

```jsx
import { Card, CardHeader, CardBody, CardFooter } from 'kui-mobile';
```

## Card

| 名称 | 类型    | 默认值 | 描述     |
| ---- | ------- | ------ | -------- |
| full | boolean | -      | 是否通栏 |

## CardHeader

| 名称  | 类型                      | 默认值 | 描述       |
| ----- | ------------------------- | ------ | ---------- |
| extra | React.ReactNode \| string | -      | 右侧操作区 |

## CardFooter

| 名称  | 类型                      | 默认值 | 描述       |
| ----- | ------------------------- | ------ | ---------- |
| extra | React.ReactNode \| string | -      | 右侧操作区 |
