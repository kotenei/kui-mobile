# API

```jsx
import { Tag } from 'kui-mobile';
```

## Tag

| 名称     | 类型          | 默认值 | 描述                                                           |
| -------- | ------------- | ------ | -------------------------------------------------------------- |
| color    | string        | -      | 颜色风格，可选值 'primary' 'info' 'success' 'warning' 'danger' |
| closable | boolean       | false  | 能否关闭                                                       |
| line     | boolean       | false  | 是否显示线框                                                   |
| onClose  | () => boolean | -      | 关闭时回调                                                     |
