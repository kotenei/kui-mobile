# API

```jsx
import { Divider } from 'kui-mobile';
```

## Divider

| 名称      | 类型   | 默认值 | 描述                                                           |
| --------- | ------ | ------ | -------------------------------------------------------------- |
| component | string | 'hr'   | 组件容器                                                       |
| color     | string | -      | 颜色风格，可选值 'primary' 'info' 'success' 'warning' 'danger' |
